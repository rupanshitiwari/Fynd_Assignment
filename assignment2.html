Q2. This exercise helps understand the difference between inline, block, and inline-block display types. On an HTML page add 3 span elements each with the words HTML, CSS, and JavaScript. Select the 3 elements using the span selector in an embedded style tag on the page and set the width to 40%.

Q1- How do elements appear?Does the width have any effect? Why/why not?

Ans- The elements appear in same line i.e. Inline. The width has no effect.This is because span is an inline tag.

Q2-- Set display to inline.Does it cause any change? why/why not.

Ans- There is no change because Span itself is an inline element.

Q3-- Change display to block.What change does it cause?

Ans- The elements now appears in verticle order.This is because the the property changes to block and block element takes up the whole width.

Q4-- Set display to inline-block.What change does it cause?

Ans- Now the elements have taken the width accordingly i.e. the HTML and CSS are in same line and the JAVASCRIPT appears in next row.

Inline element :- In inline element all the elements are placed horizontally side by side. In this type we cannot change the height and width of the element.

Block element :- In block element all elements are stacked vertically rowwise. In this type we can change the height and width of the element.

Inline-block element :- In inline-block all the elements are placed horizontally side by side but here we can specify the height and width of the element.
